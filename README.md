# Date invitation site

A small static website ready for GitHub Pages.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and the `assets` folder.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. GitHub will publish it at `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`.

## Customize

- Change the day/time buttons in `index.html`.
- Change the WhatsApp message in `script.js` inside `buildMessage()`.
- Replace `assets/invitation.png` with another image using the same filename.

The No button moves four times, then becomes fully clickable.
